EDO Studios

Aplicación web educativa para el análisis y solución de problemas que se resuelven con ecuaciones diferenciales ordinarias (EDO).

Inicio

Resumen del proyecto

La razón de crear una aplicación web se basa en las dificultades que hemos observado al momento de incluir TIC en los planes de estudio de matemáticas. Mayormente estas dificultades son debido a los altos costos de suscripción por el software y al poco tiempo que se tiene para capacitar a un grupo a usarlo. Es por tal que se propuso el desarrollo de esta app, la cual tiene un enfoque destacado en la interactividad. Creemos que con esto mejorará significativamente el rendimiento académico y motivación de los estudiantes.

Para las categorías de EDO se creó una estructura de página dividida en 3 secciones que facilita la comprensión de los temas:

  • Teoría; Sintetización de conceptos incluyendo las aplicaciones en el mundo real, fórmulas, y métodos de resolución manual para los problemas

  • Visualización; Video del maestro resolviendo un problema, detallando en el proceso para aplicar las fórmulas y el razonamiento necesario para obtener los valores del texto.

  • Práctica; Programas interactivos para la calculación sistemas no homogeneos 2x2, y evaluación de respuestas para problemas de valor inicial.
